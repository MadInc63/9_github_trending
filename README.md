# Github Trends

The script displays a list of 20 repositories for the last week of those who gained popularity.

# How to use

Run the script using Python 3.5 from the command line.
The script displays 20 references to the repositories created in the last week with the highest number of stars and the counted number of issues.

```Bash
$python github_trending.py
```
